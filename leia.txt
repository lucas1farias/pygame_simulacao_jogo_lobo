

pygame_simulacao_jogo
Simulacro de jogo em Pygame usando recursos importantes do biblioteca (superfícies, colisão, sprites animados, movimento de câmera)

Procedimento principal
pip install -r dependencies.txt

Detalhes
Esse repositório é diretamente relacionado ao repositório "pygame_criar_mapas"
Ao executar um dos arquivos em "pygame_criar_mapas", é gerado no terminal um array de strings e este array pode ser copiado e usado neste repositório "pygame_simulacao_jogo", em "settings.py"
settings.py
Temos os mapas existentes aplicáveis ao jogo (criados via repositório "pygame_criar_mapas")
Apesar dos mapas aqui serem aleatórios, é possível optar por criar mapas manualmente
A variável "scenario2" é um exemplo de um cenário manual meu
Cada array de strings contêm um "X", que representa uma superfície (escolha opcional do caracter)
Cada array de strings não vêm com um jogador, este foi adicionado manualmente via caracter "P"
Para substituir mapas, ir em "level.py", procurar por "self.setup_level" e trocar o parâmetro por um novo array de strings ou outra variável do arquivo "settings.py"
launcher.py
Executa o simulacro do jogo
